# Pass 43 — The Unobstructed Phenomenon

The Gatehouse is now a true evidence rail rather than an overlay on the observed field.

## Changes

- Opening the Gatehouse reserves a dedicated right-side reading rail on large desktop screens.
- The Leaflet map, plume canvas, atmospheric breath, evidence light, and cinematic focus contract together so the source, possible movement, and terrain remain visible.
- A one-click Compact view keeps the Field Insight and case selector while hiding secondary comparison detail.
- Small Gatehouse text was raised to a more readable baseline.
- Narrower screens retain the established overlay behavior because there is not enough room for a separate rail.
- Reduced-motion users receive the same layout without animated resizing.

## Scientific boundary

This pass changes layout and legibility only. It does not alter source geometry, wind context, terrain interpretation, evidence state, confidence, or claim authority.
