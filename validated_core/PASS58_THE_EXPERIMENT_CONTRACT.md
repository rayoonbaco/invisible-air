# PASS 58 — The Experiment Contract

## Objective
Make the parent demonstration, active experiment, atmospheric-input provenance, and model state visibly distinct without adding another panel.

## Changes
- Added an explicit Parent Demonstration line.
- Renamed the changing location state as the Active Experiment.
- Added Curated / Relocated / Visitor-added source provenance.
- Added Synchronized and Fallback Inputs integrity states.
- Kept coordinates, live atmosphere, displayed wind, and governing field on the Pass 57 authoritative chain.

## Boundary
A parent scenario is context, not a claim that its original weather drives a relocated source.
