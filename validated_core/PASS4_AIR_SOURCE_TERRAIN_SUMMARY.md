# Pass 4 — Air, Source, Terrain

## Purpose
Make the default scene read as a place with one source and uncertain air movement, not a stack of scientific raster products.

## Changes
- All rectangular scientific raster panes, including hillshade and relief overlays, are hidden in Essence mode.
- Scientific rasters remain available in Context and Science modes.
- Bottom scientific annotation chips and timeline are hidden in Essence mode.
- Map labels, uncertainty polygon, source radius, and downwind marker are hidden in Essence mode.
- The source point remains visible.
- The flow centerline is reduced to a faint dashed cue.
- The moving air canvas receives restrained contrast and brightness so the phenomenon reads without becoming a detection claim.
- Introductory copy, map control, attribution, and evidence launcher recede further.

## Scientific boundary
This remains a human-review reconstruction using available terrain and current atmospheric context. It is not direct methane detection or verified historical transport.
