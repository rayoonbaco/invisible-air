# Pass 7A - Dream With Discipline, Resolved

## Strategic goal
Preserve the dreamlike atmospheric experience while eliminating the final visual artifact that made the science look like a rectangular sheet laid over the map.

## Changes
- Added programmatic disclosure control for every scientific Leaflet pane, so Essence mode cannot render derived raster rectangles even when inline Leaflet opacity is present.
- Kept all derived layers available in Context and Science modes.
- Removed the fallback graphic once real map tiles are loaded.
- Refined the atmospheric particles into finer directional traces, reducing the glowing-dot or smoke-cloud appearance.
- Kept the source quiet and warm while allowing the path to cool, loosen, and fragment downstream.

## Scientific boundary
The scene remains an evidence-bounded human-review reconstruction. It is not direct methane detection, concentration mapping, exposure analysis, or verified historical transport.
