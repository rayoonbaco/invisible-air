# SV2-35 — Gust and Variability Window

Adds a timestamp-gated five-hour modeled wind window centered on the nearest available observation hour. It reports gust factor, sustained-wind range, maximum gust, directional spread, window completeness, temporal offset, provenance, and a bounded low/moderate/high variability screen. It does not claim measured turbulence, sub-hourly gust chronology, dispersion, or intermittent emissions.
