# SV2-42 — Integrated Terrain-Response Field

Combines the validated DEM-derived response surfaces into one auditable synthesis with four regimes: confined, transfer, dispersive, and mixed-transition. The field preserves component support, spatial agreement, disagreement, confidence gating, provenance, and safe-unavailable behavior.

It is not measured airflow, CFD, plume transport, methane concentration, exposure, or proof of actual atmospheric behavior.
