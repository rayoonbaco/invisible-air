# Data Boundaries

## Safe language

- reported methane signal
- source-seeded observation
- visual reconstruction
- current wind context
- terrain context
- human review only

## Not claimed

- methane detection by this app
- exact plume-product geometry
- certified emissions quantification
- facility attribution
- enforcement readiness
- illegal activity

## Visual plume boundary

The animated plume is a wind-informed visual reconstruction unless exact time-sequenced methane product data is loaded.
