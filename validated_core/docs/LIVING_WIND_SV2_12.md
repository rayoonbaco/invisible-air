# SV2-12 — Living Wind

This pass makes current wind visible through bounded motion rather than arrows. Live wind speed changes particle cadence and spacing. A slow gust cycle changes velocity slightly, cross-plume sway keeps the air body alive, sparse turbulence pockets interrupt uniform motion, and measured terrain can add a restrained downstream wake cue.

## Evidence boundary

These are visual responses to current wind and measured terrain context. They are not a measured turbulence field, observation-time wind reconstruction, atmospheric dispersion simulation, methane concentration, plume height, emissions rate, or exact source-product geometry.
