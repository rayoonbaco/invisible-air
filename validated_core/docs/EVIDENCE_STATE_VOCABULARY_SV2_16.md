# SV2-16 — Evidence-State Vocabulary

This pass defines a controlled evidence ontology for every visible layer. Each layer receives exactly one primary state plus a purpose, what it represents, and what it does not represent.

Allowed states: observed, measured, modeled, current context, visual reconstruction, contextual, inferred, unavailable, unknown, not modeled, and not claimed.

The vocabulary classifies evidence; it does not increase certainty or validate providers.
