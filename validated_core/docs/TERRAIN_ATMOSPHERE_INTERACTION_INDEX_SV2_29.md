# SV2-35 — Terrain–Atmosphere Interaction Index

This pass synthesizes measured terrain anatomy, terrain confidence, and available atmospheric context into a bounded review index.

## Components
- terrain steering
- terrain shelter
- channeling potential
- ridge exposure
- vertical mixing opportunity when observation-time stability and boundary-layer evidence exist

The composite renormalizes around available inputs and explicitly records whether it uses current atmospheric context only or observation-time context.

## Boundary
The index is not a dispersion model, methane transport simulation, concentration estimate, source attribution, emissions quantification, or proof of actual airflow behavior.
