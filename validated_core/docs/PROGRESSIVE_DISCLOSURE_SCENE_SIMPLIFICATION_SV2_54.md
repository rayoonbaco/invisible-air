# SV2-54 — Progressive Disclosure and Scene Simplification

SV2-54 reorganizes the completed science stack into three reviewer-controlled visibility levels: Overview, Context, and Deep audit.

The default scene retains the minimum complete scientific story: reported observation, measured terrain, current wind context, integrated response, confidence, uncertainty, and missing-evidence status. Supporting layers remain available in Context. All technical routes and machine-readable contracts remain available in Deep audit.

No scientific layer is deleted and no evidence state is upgraded by the visibility system.

## Scientific boundary

Progressive disclosure changes visibility and reading order only. It does not remove evidence, increase confidence, detect methane, reconstruct transport, quantify emissions, or attribute responsibility.
