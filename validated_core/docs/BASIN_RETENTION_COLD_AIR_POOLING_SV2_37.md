# SV2-37 — Basin Retention and Cold-Air Pooling Potential

Adds confidence-gated, map-registered basin-retention and conditional cold-air-pooling visual potentials derived from the validated DEM. Atmospheric support is used only when observation-time stability evidence exists.

## Boundary
These layers are not measured temperature inversions, verified pollutant trapping, observed cold-air drainage, CFD, methane concentration, exposure, or proof that emissions accumulated in a basin.
