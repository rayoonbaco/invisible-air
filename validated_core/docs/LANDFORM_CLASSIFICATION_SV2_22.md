# SV2-27 — Landform Classification

This pass derives reproducible terrain classes from the validated continuous DEM using multi-scale topographic position, slope, local relief, and curvature. It produces a class raster, a bounded confidence raster, class distribution metadata, and a review page.

Classes include flat, ridge, spur, valley, drainage, saddle, upper slope, midslope, lower slope, bench, basin, and shoulder. These are geomorphometric categories, not field-surveyed geology. They carry no methane, atmospheric transport, or attribution meaning.
