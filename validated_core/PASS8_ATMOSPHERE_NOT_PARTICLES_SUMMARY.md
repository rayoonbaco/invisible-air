# Pass 8 - Atmosphere, Not Particles

## Purpose
Replace glitter-like particles with a layered atmospheric field while preserving the evidence-bounded path and all scientific modes.

## Changes
- Added extremely subtle full-scene ambient drift so the whole viewport feels atmospheric.
- Added eleven fine terrain-guided field ribbons that read as air rather than discrete objects.
- Reduced Essence-mode particle count, point size, glow, and opacity.
- Preserved downstream fading and fragmentation as the visual language of weakening support.
- Preserved Context and Science modes and all underlying scientific contracts.

## Design rule
The viewer should perceive lifting, flow, folding, and dissipation before noticing individual particles.
