# Pass 14 - Addition by Subtraction

## Philosophical test
The team removed more than half of the visible atmospheric machinery in Essence mode and asked whether the terrain became easier to read.

## Strategic result
The landscape is now the primary narrator. Atmosphere appears only as a broad breath, a fine evidentiary thread, and restrained terrain illumination. The visual choirs from earlier passes remain available in code and in deeper review modes, but stay silent in Essence.

## Changes
- Removed Visual Symphony, atmospheric veils, and segmented-field layers from the default Essence composition.
- Reduced ambient air, possibility-field, volume, particle, source, and centerline prominence.
- Increased terrain legibility with slightly brighter relief and calmer cartography.
- Preserved the 30-second Second Movement, now choreographing attention rather than visual density.
- Preserved all Context, Science, evidence, citations, provenance, uncertainty, and audit systems.

## Scientific boundary
The remaining light and movement represent evidence-bounded terrain-response tendencies. They are not measured transport, methane concentration, exposure, or verified plume geometry.

## Acceptance sentence
The viewer should understand more because the project shows less.
