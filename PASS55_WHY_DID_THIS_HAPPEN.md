# PASS 55 — Why Did This Happen?

The Atmospheric Observatory now publishes a second deterministic sentence after a source move.

- **Model Observation** states what changed.
- **Why the Model Changed** explains the change using only the geometry and diagnostics already produced by the governing model.

The explanation may reference source separation, cross-corridor alignment, bounded terrain steering, modeled spread, supported footprint, or unchanged governing assumptions. It does not use generative AI and does not claim measured concentration, exposure, forecast behavior, or chemical interaction.

No new tab, card, assistant, or settings panel was added.
