# PASS 61 — Navigation Clearance

## Objective
Keep the Return to Observatory control fully visible and comfortably separated from the upper-left flag/banner treatment.

## Changes
- Moved the control lower and slightly inward on desktop.
- Added a restrained dark backing surface and border so the control remains legible over map imagery.
- Preserved a 44-pixel touch target on smaller screens.
- Increased stacking priority so decorative layers cannot cover it.

## Scope
No scientific model, renderer, journal, source behavior, or experiment state was changed.
