# Pass 32 — The Atlas Speaks Clearly

## Purpose
Strengthen the Observation Atlas without returning to cards, dashboards, or fabricated cases.

## Changes
- Enlarged the Atlas heading, lead sentence, case titles, questions, and scientific detail.
- Reframed each observation around four concise comparison dimensions: Land, Air, Time, and Trust.
- Added visible evidence-package readiness for every case.
- Added a compact Atlas status line showing verified scenes, reserved comparisons, and required evidence fields.
- Replaced vague reserved-case language with the exact number of verified fields still required.
- Preserved the rule that unavailable cases cannot render invented atmospheric scenes.

## Scientific boundary
The Atlas compares the completeness and interpretive context of case packages. It does not imply that reserved cases are observations, detections, verified plume paths, concentrations, exposure estimates, or attributions.
